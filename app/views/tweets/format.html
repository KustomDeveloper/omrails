Summary

These are the most common Template tags used in Wordpress Development. It is not intended to be a complete list of Wordpress functions, just the common ones for someone just getting started developing with Wordpress. Many of the template tags are meant to be used within the Wordpress loop but I have also included some like login/logout, calendar and some other ones that are commonly placed in the sidebar or footer.
